# HireMe API

A Job Application Management Backend built with FastAPI and TiDB Cloud.

## Features
- JWT Authentication
- Role-based Access (Admin/User)
- Job Posting System
- Job Applications
- Secure Password Hashing (Argon2)
- Cloud Database (TiDB with SSL)

## Tech Stack
- FastAPI
- SQLAlchemy
- TiDB Cloud (MySQL-compatible)
- Argon2
- JWT
